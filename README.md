# VaultFlow

VaultFlow é um app web de controle financeiro com foco em UX estilo fintech, multiusuário e visão agregada do casal.

## Acesso
- App (GitHub Pages): https://hbtmarc.github.io/vaultflow/

## Estrutura
- /docs -> versão publicada no GitHub Pages
- /app  -> código-fonte do aplicativo (desenvolvimento)
- /firebase -> regras e configurações do Firebase (quando configurado)

## Ambientes
- dev / stg / prod (Firebase)

## Roadmap (MVP)
- Auth (login/logout)
- Wallets (Marcelino, Luiza, Casal agregado)
- Transações / categorias
- Dashboard mensal
